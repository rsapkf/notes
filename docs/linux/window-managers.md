## dwm
- Write a desktop entry for dwm so that it shows up as an option in your login
  screen on GDM on Ubuntu:

  ```
  # /usr/share/xsessions/dwm.desktop
  [Desktop Entry]
  Encoding=UTF-8
  Name=Dwm
  Comment=Dynamic window manager
  Exec=dwm
  Icon=dwm
  Type=XSession 
  ```
